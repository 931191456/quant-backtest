# 量化回测系统

基于 Streamlit 的 A 股量化回测网站，支持 ETF、个股的技术指标回测分析。

## 功能特性

- **标的丰富**：支持 ETF、个股、指数的回测
- **策略多样**：均线交叉、MACD、RSI、KDJ、布林带、缩量反转、恐慌抄底等策略
- **参数可调**：支持自定义均线周期、止盈止损、最大持仓天数等
- **图表可视化**：交互式 K 线图、资金曲线、回撤图、月度收益热力图
- **统计全面**：总收益、年化收益、最大回撤、胜率、盈亏比等核心指标
- **数据导出**：支持导出交易明细和回测报告

```

## Streamlit Cloud 部署

1. 将项目文件上传到 GitHub 仓库
2. 访问 [Streamlit Cloud](https://streamlit.io/cloud)
3. 点击 "New app" 创建新应用
4. 选择你的 GitHub 仓库和分支
5. 设置主文件路径为 `app.py`
6. 点击 "Deploy!"

部署后即可通过云端访问你的回测系统。

## 使用说明

### 1. 标的设置
- 选择标的类型（ETF 或个股）
- 输入标的代码（6位数字）
  - ETF示例：515000（科技ETF）、510300（沪深300ETF）
  - 个股示例：000001（平安银行）、600519（贵州茅台）

### 2. 回测参数
- 设置初始资金（默认10万元）
- 选择回测时间范围
- 设置买卖费率（默认买入万2.5，卖出万2.5+千1印花税）
- 选择基准对比指数

### 3. 策略选择
支持以下策略：
- **均线交叉策略**：短期均线上穿长期均线买入，下穿卖出
- **MACD策略**：DIF与DEA金叉买入，死叉卖出
- **RSI策略**：RSI从超卖区回升买入，从超买区回落卖出
- **布林带策略**：价格触及下轨买入，触及上轨卖出
- **KDJ策略**：K值从下往上穿越D值买入，反之卖出
- **缩量反转策略**：放量下跌后缩量反弹时买入
- **恐慌抄底策略**：大幅下跌且放量时买入

### 4. 风险管理
- 止损比例：达到亏损比例自动止损
- 止盈比例：达到盈利比例自动止盈
- 最大持仓天数：持仓超过此天数自动平仓

### 5. 查看结果
- **K线图**：显示价格走势、均线、买卖点标记
- **资金曲线**：显示策略净值 vs 基准净值
- **回撤曲线**：显示每日回撤情况
- **交易分布**：显示盈利/亏损分布
- **月度收益**：热力图展示每月收益
- **交易明细**：完整的交易记录列表

## 项目结构

```

├── app.py              # Streamlit主应用
├── data_fetcher.py     # 数据获取模块（akshare）

## 测试数据接口

首次使用前，建议先运行测试脚本验证数据接口是否正常：

```bash
cd 量化回测网站
python test_data.py
```

如果测试通过，说明数据接口工作正常，可以正常使用回测功能。

## 数据来源

本系统使用 [akshare](https://akshare.akfamily.xyz/) 库从东方财富获取数据：
- ETF数据：`fund_etf_hist_em`
- 个股数据：`stock_zh_a_hist`
- 指数数据：`index_zh_a_hist`

akshare 是免费、开源的中国金融数据接口，数据来自东方财富等权威平台。
